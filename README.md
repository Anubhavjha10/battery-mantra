# Battery Mantra

This is a modern React web application built with [Vite](https://vitejs.dev/).

## Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```
2. **Start the development server**:
   ```bash
   npm run dev
   ```
3. **Build for production**:
   ```bash
   npm run build
   ```

## Architecture

- **`public/assets`**: Contains static assets like images, fonts, and template CSS files.
- **`src/`**: Contains the React source code.
- **`src/context`**: Contains React Context providers for global state (Cart, Language, Currency, Location).
- **`src/utils`**: Contains utility functions and helpers.
